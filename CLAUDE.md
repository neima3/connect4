# Connect4

## Project Overview
Create a web based connect Connect 4 game that looks beautiful and functions well in all browsers (mobile included) and has option to play against computer or to play against somebody online using a shared code. make the animations/transitions/icons very beautiful, modern and professional. add any features you think would be good.

## Tech Stack
nextjs

## Development Server
- Port: 4002

## Development Guidelines
- Do not use browser confirm() or alert() dialogs.
- Always create a help section with app instructions and features.
- Use TypeScript for all code.
- Follow accessibility best practices (semantic HTML, ARIA labels, keyboard navigation).
- Mobile-first responsive design.
- Use proper error handling and user-friendly error messages.
- Include proper loading states and skeletons.
- Add proper meta tags and SEO.

## Code Style
- Use TypeScript for all code
- Follow the existing project structure
- Keep components small and focused
- Use meaningful variable and function names
- Add comments only for complex logic

## Important Rules
- Do not modify files outside the project directory
- Always test changes before considering a step complete
- If a step fails, document the error and attempt to fix it
- Use git commits with meaningful messages after each significant change
